  2. 堆栈内存空间是多大？ ---- 没回答上，但还是瞎扯了一下

    • 自己的回答

      堆是存储的变量

    • Max回答

      现在还是不会，感觉百度这个也不太像是考点一样。

    • 百度的回答

      栈空间：自动内存空间，其中数据的大小在编译时确定，数据的分配和释放也由编译器在函数进入和退出时插入指令完成，数据生命周期和函数一样。

      堆空间：动态（手动）内存空间，其中数据的大小和初始值在运行时确定，数据生命周期不定。

      linux 命令 ulimit -a 可以看到stack size (kbytes) 8192 说明Linux默认的用户栈空间的大小是8MB
      堆空间来说，默认是没有软限制的

  4. 谈谈对于channel的理解？---- 算是回答上了的，扩展了一些

    • 自己的回答

      golang协程的chan,是线程中启动的协程，独立运行且占用很少的系统资源 --- 你这个回答怕是错的哦

    • Max回答

      channel是Go语言独有的，不同于其他语言，它的设计初衷是想要以通信的方式来共享变量，其他语言是以共享变量的方式来通信，channel可以做到一些事，比如控制打印顺序，监听信号量来控制程序的走向等，并且channel分为有缓冲和无缓冲两种，无缓冲的必须发送和接收方都要有，否则就会产生dead lock；有缓冲的允许往里面发送数字小于设定值的数据，如果大于了，也会dead lock。如果关闭了channel，还能从channel中读数据，不过要判断是否ok， 但是如果还往channel写数据，会panic。

    • 百度的回答

      chan是golang协程进行通信的通道，简单的用 <- ch（读） 和 ch <-（写）数据，
      比传统线程通信（共享内存、信号量）更简单且是用户级别的调度模块

  11. socket的粘包怎么解决？ ---- 没得丝毫头绪

    • 自己的回答

      不会

    • Max回答

      不会，现在都还没去看呢

    • 百度的回答

      1. 发送方和接收方规定固定大小的缓冲区，也就是发送和接收都使用固定大小的 byte[] 数组长度，当字符长度不够时使用空字符弥补；
      2. 在 TCP 协议的基础上封装一层数据请求协议，既将数据包封装成数据头（存储数据正文大小）+ 数据正文的形式，这样在服务端就可以知道每个数据包的具体长度了，知道了发送数据的具体边界之后，就可以解决半包和粘包的问题了；
      3. 以特殊的字符结尾，比如以“\n”结尾，这样我们就知道结束字符，从而避免了半包和粘包问题（推荐解决方案）。

  13. 3块钱一瓶饮料，3个空瓶子又可以换一瓶饮料，给你15块钱，你一共能喝多少瓶饮料， 如果是N块钱呢。 5分钟内代码写出来。 ---- 最后写出一个递归的。 但是好像base case判断是错的。 算不上完全正确吧。 只能说思想是对的。其实时间多一点的话，这个简单，写出来完全没问题，感觉没有做对，有点亏。

    • 自己的回答

      大概思路

      func A -> N除以三取余 得值 a 余数 b
      总瓶数 += a

      a+b 再去调 func A 递归
      最后a+b < 3 则不能再换了 得出总数

    • Max回答

      我回来后整理了两种方式，面试的时候，写的第一种

      //test
/**
这个函数是解决多少空瓶，换多少个新瓶的问题
 */
func test(totalNum int, rest int) int {
	if (totalNum + rest) < 3 {
		return 0
	}
	exchangeNum := int(math.Floor(float64((totalNum + rest) / 3)))
	restNum := int(math.Floor(float64((totalNum + rest) % 3)))
	return exchangeNum + test(exchangeNum, restNum)
}

func main() {
	totalMoney := 150
	initialNum := int(math.Floor(float64(totalMoney / 3)))
	a := initialNum + test(initialNum, 0)
	fmt.Println(a)
}

      //test2
/**
这个函数是迭代，直接把总钱数传入，得到所有瓶子的
 */
func test2(totalMoney int) int {
	exchangeNum := int(math.Floor(float64(totalMoney / 3)))

	allRestNum := exchangeNum
	for allRestNum >= 3 {
		tmpNum := int(math.Floor(float64(allRestNum / 3)))
		rest := int(math.Floor(float64(allRestNum % 3)))
		allRestNum = tmpNum + rest
		exchangeNum += tmpNum
	}
	return exchangeNum
}
func main() {
	totalMoney := 150
	b := test2(totalMoney)
	fmt.Println(b)
}
